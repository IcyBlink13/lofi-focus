Lo-Fi Focus — веб-платформа для самостійного навчання грі на музичних інструментах у стилі lo-fi. Короткі мікроуроки, фазовий таймер практики, персональні маскоти-гіди та система мотивації у вигляді стріків і досягнень.
🔗 Живий сайт: icyblink13.github.io/lofi-focus
